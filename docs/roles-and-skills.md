## Roles and responsibilities

- Mobile engineer
  - Develops and maintains mobile clients and integrates backend APIs.
- Back-end engineer
  - Implements server-side logic and APIs, ensures scalability.
- DevOps / SRE
  - Deploys, monitors, and maintains production systems.
- Security engineer
  - Designs and reviews authentication, encryption, and security controls.
- QA engineer
  - Tests functionality, reliability, and edge cases.
